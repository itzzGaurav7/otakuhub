## Installation

Use the following command in both the directories i.e Client and Server

```bash
npm install
```
For Server, add a .env file with following data
```bash
MONGO_URL=<mongo url>
JWT_KEY=<secret key>
PORT=<port to run>

```
Start Server
```bash
npm start
```
Start React App
```bash
npm run dev
```
